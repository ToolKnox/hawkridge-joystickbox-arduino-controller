# Assembly guide — JoystickBox Arduino Controller

This Hawkridge remix package is prepared as a **remix / variation** of the original source project.

Original source: https://github.com/agomezgar/joystickBox

## Files to print / review

- `STL__cajaJoystick.stl`
- `STL__tapaCajaJoystick.stl`

## Suggested assembly flow

1. Review the staged printable/CAD files and choose the newest matching part set before printing.
2. Print structural parts first, then smaller covers/buttons/brackets once the main fit is confirmed.
3. Dry-fit the printed pieces before installing electronics, bearings, motors, or fasteners.
4. Install electronics and wiring only after mechanical travel is clear and no printed part binds.
5. Keep the upstream source mirror available from GitHub so builders can compare against original diagrams and firmware/source notes.

## Documentation download

The public GitHub documentation repo is the download point for the README, build notes, source mirror, firmware/PCB/CAD assets, and Bill of Material documentation.
