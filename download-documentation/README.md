# JoystickBox Arduino Controller — download documentation

Public documentation/source repository: https://github.com/ToolKnox/hawkridge-joystickbox-arduino-controller

## Download everything from GitHub

Use the green **Code** button on the GitHub repository, then choose **Download ZIP**. The ZIP/clone contains:

- the README build/install guide
- user-facing docs and build notes
- firmware/software/PCB/CAD/source assets where available
- mirrored upstream source files under `source/upstream/`

## Printable files

Download the printable model files from the Printables Files section: https://www.printables.com/model/1776633-joystickbox-arduino-controller

## Suggested starting point

1. Open `README.md` in this repository.
2. Read the build/install notes before printing or powering electronics.
3. Use the Bill of Material PDF attached to the marketplace Documentation section for parts planning.
